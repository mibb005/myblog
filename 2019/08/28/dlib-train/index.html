dlib深度学习模型训练

发表于 2019-08-28 | 更新于 2021-06-21 | 评论数：

下载

https://github.com/davisking/dlib

win10 编译imglab

  • 进入tools\imglab目录
  • 新建文件build，进入build
  • 右键cmd命令，输入：cmake ..
  • cmd命令，输入：cmake --build . --config Release
  • 将Release目录加入到系统环境变量

编译完成

imglab图片标注

  • 新建文件夹：images
  • 在images下放你想标注的图像，就是训练集
  • 输入：imglab -c mydataset.xml images，生成mydataset.xml文件
  • cmd输入：imglab mydataset.xml
  • 出现imglab标注软件了，可以自己标注了，这是一个漫长的过程
  • 打标注，打标签，保存，退出软件

xml其实就是保存标注点的坐标信息了
